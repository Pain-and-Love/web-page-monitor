## a mono repo.  
`web-page-monitor` is a website, for end users to use. created by Next.js  
`pptr` is the crawler.  
`socketio` is the data communication center.  

## Intro  
`web-page-monitor` 最终用户使用的网站界面  
`pptr` puppeteer执行程序 
`socketio` 信息服务中转站

## ports
Next.js 3002  
Static HTML 3001 on production.
socket.io server 3003
