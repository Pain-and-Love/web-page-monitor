## 文字变动对比 {#text-diff}

<ImageChanger style={{maxHeight: '60vh'}} src="/img/faq/email_looks_like/1-zh.png" />

:::tip  

如上图所示，邮件包含如下信息：  

任务名称(别名) 及 页面变动的时间；  
增加的内容，文字有下划线，背景色为黄绿色；  
删除的内容，文字有删除线，背景色为红粉色。  

:::

:::tip  

当页面文字过长(大于2000个字符)时，邮件内可能不会包含具体的变动对比信息。  
您需要手动回到您检测的页面，查看页面的变动。  

:::
