## Gitee.com 登录 {#gitee-login}
### 如果你已经有了Gitee.com的账号，且已绑定邮箱:  

点击 “使用Gitee.com OAuth登录”，
在弹出的窗口中，保持两个√都勾选上，然后点击“同意授权”。

> 如图所示：
<ImageChanger src="/img/faq/reg_n_login/1-zh.png" />

然后就会跳转到本网站的个人中心页面。如图所示：
<ImageChanger src="/img/faq/reg_n_login/2-zh.png" />  

:::tip
如果上述操作之后没有出现个人中心页面，请检查一下Gitee.com是否未绑定邮箱，未绑定邮箱会无法登录本网站。  
请去**[这个链接](https://gitee.com/profile/emails)**绑定邮箱，然后尝试再次登录。

:::

### 如果Gitee.com的账号，未绑定邮箱:  
请去**[这个链接](https://gitee.com/profile/emails)**绑定邮箱，然后尝试再次登录本网站。

### 如果你没有Gitee.com的账号：  
可以按照网站提示，或者点击[这里](https://gitee.com/signup#lang=zh-CN)注册，建议使用电子邮箱Email方式注册。  
:::tip
注意，输入电子邮件注册即可，不建议用手机号注册。  
:::  

然后回[网站首页](https://monit.or.passby.me/)，点击“使用Gitee.com OAuth登录”，即可。  
:::tip
如果没有反应，请查看上面未绑定邮箱的情况。  
如果Gitee绑定了邮箱，仍然无法登录，请联系我们。
:::

## Github/谷歌 登录 {#firebase-login}  
如果您在海外，您应该可以看到这两个登录方式。  
这两种登录方式会经由Firebase中转，我们仅获取您的邮箱地址，请放心使用。
