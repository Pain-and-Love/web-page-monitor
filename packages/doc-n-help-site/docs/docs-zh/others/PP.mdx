<h2 id="PrivacyPolicy"> 隐私政策 </h2>
<p className={styles.para}>
  我们(网页变动检测系统的开发及运营人员)尊重和保护用户的隐私。<br/>
  本隐私政策将告诉您我们如何收集和使用有关您的信息，以及我们如何保护这些信息的安全。<br/>
  您成为我们的用户前务必仔细阅读本隐私条款并同意所有隐私条款。<br/>
  本隐私政策条款在您登录、成为我们的用户后立即生效，并对您及我们产生约束力。 <br />

  <h3>一、登录时提供的信息</h3>
  <p>
    您在登录时，我们仅收集并存储您的Email地址。
  </p>
  <h3>二、Oauth提供方收集的信息</h3>
  <p>
    我们使用了Gitee.com、Github、Google的Oauth登录方式，使您免注册即可一键登录我们的服务。 
    其中， Github、Google的Oauth登录经由Firebase中转提供。 <br />
    因此，Gitee.com、Github、Google、Firebase均可能收集其它信息，请参阅他们的隐私政策。 <br />
  </p>
